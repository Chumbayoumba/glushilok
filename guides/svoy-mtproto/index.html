Как поднять собственный MTProto-прокси для Telegram на VPS через Docker с маскировкой FakeTLS — обход блокировки и глушилок

ЧТО ТАКОЕ MTPROTO ПРОКСИ

MTProto-прокси — это родной прокси Telegram, который маскирует трафик мессенджера. В режиме FakeTLS он притворяется обычным HTTPS-сайтом, что усложняет блокировку. Свой прокси работает только для Telegram (не для всего интернета), подключается в один тап по ссылке и не требует приложений. Минус: в 2026 ТСПУ научились частично распознавать MTProto по «рукопожатию», поэтому прокси лучше держать на свежем IP и менять при блокировках.

ПОШАГОВАЯ ИНСТРУКЦИЯ

1 Арендуй VPS вне России

Нужен дешёвый VPS (1 vCPU, 1 ГБ RAM хватит) в стране, где Telegram не заблокирован. Нидерланды, Германия, Финляндия — отличный выбор.

  • Минимум: 1 ГБ RAM, Ubuntu 22.04 / Debian 12
  • Открой порт (например 443) в фаерволе провайдера
  • Цена ~200–300 ₽/мес

2 Установи Docker

Проще всего поднять MTProto через Docker — не нужно компилировать вручную.

apt update && apt install -y docker.io
systemctl enable --now docker

3 Запусти MTProto-прокси (mtg) с FakeTLS

Сгенерируй секрет с маскировкой под крупный сайт и запусти контейнер. mtg — современная быстрая реализация на Go.

# сгенерировать secret с FakeTLS под google.com
docker run --rm nineseconds/mtg:2 generate-secret google.com
# запустить прокси на порту 443
docker run -d --name mtg --restart=always -p 443:3128 \
  nineseconds/mtg:2 simple-run -n 1.1.1.1 0.0.0.0:3128 ТВОЙ_СЕКРЕТ

4 Получи ссылку для подключения

mtg выводит готовую ссылку вида tg://proxy?server=...&port=...&secret=... Скопируй её.

  • Ссылка содержит IP сервера, порт и секрет с маскировкой
  • Можно сделать удобную ссылку через @MTProxybot в самом Telegram
docker logs mtg
# найди строку tg://proxy?server=IP&port=443&secret=ee...

5 Подключись в Telegram

Открой ссылку tg://proxy на телефоне с установленным Telegram — мессенджер сам предложит подключить прокси. Подтверди.

  • Значок прокси загорится в Telegram = работает
  • Раздай ссылку друзьям — прокси общий
  • При блокировке IP пересоздай VPS и сгенерируй новый секрет

ЧАСТЫЕ ВОПРОСЫ
MTProto прокси открывает YouTube и другие сайты?
Нет. MTProto работает только для Telegram. Для YouTube, WhatsApp и всего интернета нужен VPN на VLESS Reality.
Почему мой прокси быстро отваливается?
В 2026 ТСПУ распознают MTProto по TLS-рукопожатию и блокируют IP. Держи прокси на свежем IP, меняй сайт-маскировку FakeTLS, либо используй авто-выдачу свежих прокси в нашем боте.
Нужен ли домен?
Нет, MTProto работает по IP. Домен нужен только для некоторых TLS-схем, но для базового FakeTLS достаточно IP сервера.
Это законно?
Использование прокси для личного доступа к Telegram в России законно. Реклама средств обхода — отдельная история, её мы не касаемся.
